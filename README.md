# RubyChina

[RubyChina](https://ruby-china.org) is an online developer community with a focus on Ruby. This repo contains an unofficial source code for RubyChina App for Android.

## Thanks

* Much thanks to the RubyChina RESTful API and it's nicely [document](https://ruby-china.org/api).
* Inspired by many useful opensource projects, see details in this project.
